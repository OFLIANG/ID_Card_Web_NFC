# 更新日志

本文件记录 LeongBro_IDCard 项目的所有重要更新。

---

## v2.0 (2026-06-24)

### 新增
- 🛰️ **卫星系统简化为 7 颗**：移除真实卫星定义和 Celestrak API 联网逻辑，改为 7 颗装饰卫星均匀分布在不同轨道平面（6 个均匀倾角 + 1 随机补充，高度 500-900km）
- 📊 **遥感数据条通用化**：卫星编号使用 SAT-01 ~ SAT-07 通用编号，高度随机变化

### 移除
- 🗑️ **Celestrak 实时卫星数据**：删除 TLE 轨道数据获取和在线卫星位置解析，项目完全离线运行
- 🗑️ **10 颗真实卫星定义**：GF-05、ZY-3、HY-1C 等真实卫星名称不再使用

### 改进
- 📝 **README 重写**：新增 3D 地球组件详细说明（线框球体、大气辉光、国家边界、中国增强、定位标记、数据点、卫星系统），更新功能特性和技术参数
- 🪟 **毛玻璃还原**：`--surface` 恢复为半透明 `rgba(10, 18, 40, 0.65)` 配合 `backdrop-filter: blur(24px)`

---

## v1.9 (2026-06-24)

### 新增
- 🗺️ **国家边界线**：从 CDN 动态加载 Natural Earth 110m TopoJSON 数据，在地球上绘制 177 个国家的边界轮廓（青色半透明线条）
- 📍 **当前位置定位标记**：使用浏览器 Geolocation API 获取用户位置，在地球上放置橙色星形标记 + 脉冲雷达圈 + 垂直信标射线
- 🔆 **边界线拖拽增强**：拖拽地球时边界线透明度自动提升，松手后恢复
- 🌐 **topojson-client 依赖**：引入 topojson-client CDN 用于 TopoJSON → GeoJSON 转换

## v1.8 (2026-06-24)

### 新增
- 🌍 **地球手动拖拽旋转**：当卡片透明度调至 0%（纯地球模式）时，可直接在画面上拖拽旋转地球，支持上下左右自由翻滚
- 🎯 **拖拽惯性**：松手后地球保持惯性旋转，缓慢减速直至自动恢复自转
- ✨ **大气层辉光反馈**：拖拽时大气层辉光增强并变色（蓝→青），松手后平滑恢复
- 📐 **触摸震动反馈**：移动端触摸地球时触发轻微震动

### 修复
- 📐 **缩放范围精确化**：缩放仅作用于卡片内容区域（`.card-container`），悬浮按钮、缩放控制面板、滚动条等 UI 控件不再受缩放影响，始终固定在页面边缘
- 🎾 **网球弹窗二维码**：使用实际微信二维码图片 `img/WX.jpg` 替代动态生成的 QR 码，图片嵌入圆角矩形框内展示

## v1.6 (2026-06-24)

### 新增
- 🌍 **地球同步缩放**：缩放时 Three.js 地球背景同步缩放（通过 `camera.zoom`），与 UI 保持一致的视觉比例

### 修复
- 🔒 **透明度0禁止交互**：透明度为 0% 时卡片 `pointer-events: none`，点击原有元素位置不会触发任何跳转链接

## v1.5 (2026-06-24)

### 修复
- 🐛 **缩放居中修复**：改用 CSS `transform: scale()` 替代 `body.style.zoom`，缩放时 3D 地球背景始终相对界面居中显示
- 🐛 **透明度作用范围修复**：清除 CSS `cardReveal` 动画对 inline `opacity` 的覆盖，透明度滑块现在能正确控制卡片内所有元素（文字、图标、头像等）
- 🐛 **透明度滑轨方向修复**：移除 `updateOpacityFromPointer` 中的 `1 - ratio` 反转，滑动方向恢复正常（右=100%，左=0%）

## 版本规范

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：

- **主版本号 (MAJOR)**：不兼容的 API 修改
- **次版本号 (MINOR)**：向下兼容的功能性新增
- **修订号 (PATCH)**：向下兼容的问题修正
