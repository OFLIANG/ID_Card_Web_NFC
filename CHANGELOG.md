# 更新日志

本文件记录 LeongBro_IDCard 项目的所有重要更新。

---

## v1.7 (2026-06-24)

### 修复
- 📐 **缩放范围精确化**：缩放仅作用于卡片内容区域（`.card-container`），悬浮按钮、缩放控制面板、滚动条等 UI 控件不再受缩放影响，始终固定在页面边缘

---

## v1.6 (2026-06-24)

### 新增
- 🌍 **地球同步缩放**：缩放时 Three.js 地球背景同步缩放（通过 `camera.zoom`），与 UI 保持一致的视觉比例

### 修复
- 🔒 **透明度0禁止交互**：透明度为 0% 时卡片 `pointer-events: none`，点击原有元素位置不会触发任何跳转链接

## v1.5 (2026-06-24)

### 修复
- 🐛 **缩放居中修复**：改用 CSS `transform: scale()` 替代 `body.style.zoom`，缩放时 3D 地球背景始终相对界面居中显示
- 🐛 **透明度作用范围修复**：清除 CSS `cardReveal` 动画对 inline `opacity` 的覆盖，透明度滑块现在能正确控制卡片内所有元素（文字、图标、头像等）
- 🐛 **透明度滑轨方向修复**：移除 `updateOpacityFromPointer` 中的 `1 - ratio` 反转，滑动方向恢复正常（右=100%，左=0%）

---

## v1.4 (2026-06-24)

### 新增
- 🎾 **网球助教弹窗**：海报展示 + 课程信息卡片 + 微信二维码 + 电话/微信咨询按钮
- 🎨 **网球图标统一色调**：SVG 矢量图标，使用 `currentColor` 与 FA 图标统一青色
- 🔍 **透明度滑块优化**：滑轨加长至 200px，阈值扩展至 0%-100%（0% 时仅显示背景地球）

---

## v1.3 (2026-06-24)

### 新增
- 💧 **透明度调节功能**：FAB 悬浮菜单新增透明度滑块，可将信息卡片调至完全透明，欣赏背景地球

### 改进
- 🌐 **背景地球放大**：线框球体 1.5→2.0，大气层 1.7→2.2，视觉冲击力更强
- 📐 **缩放居中修复**：缩放操作后自动滚动至居中位置，地球与卡片始终居中

---

## v1.2 (2026-06-23)

### 新增
- 🎾 **网球助教入口**：数据链路区新增网球助教链接，点击弹出详情弹窗

### 改进
- 📱 **四列社交网格**：数据链路区改为 4 列等宽排列
- 📊 **遥感数据条优化**：顶部卫星状态栏 + 语言切换按钮

---

## v1.1

### 新增

### 改进
- 📐 **屏幕自适应**：自动检测设备屏幕尺寸，小屏自动缩放适配

## v1.0

### 新增
- 🌍 **Three.js 3D 地球**：线框地球 + 星空 + 大气辉光，支持触摸/鼠标交互旋转
- ☄️ **流星粒子效果**：鼠标轨迹产生流星尾迹粒子，背景有持续的流星划过
- 📲 **保存联系人**：一键下载 `.vcf` 文件，自动添加到手机通讯录
- 📤 **分享名片**：支持 Web Share API，一键分享到社交平台
- 🌐 **中英文切换**：一键切换中英文界面，偏好自动保存至 localStorage
- 📳 **触觉反馈**：手机端支持震动反馈（点击、滚动、保存成功三种模式）
- 🪟 **玻璃拟态 UI**：半透明磨砂卡片 + 渐变发光效果 + 遥感数据条
- 🔗 **社交链接**：GitHub、抖音、网易云音乐一键跳转
- 📱 **移动端优先**：针对 NFC 扫描场景，完美适配手机端
- 💚 **微信添加**：点击弹出微信弹窗，含二维码 + 一键复制微信号 + 跳转微信 App
- 🎨 **动态入场动画**：卡片逐层淡入 + 扫描线覆盖层

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：

- **主版本号 (MAJOR)**：不兼容的 API 修改
- **次版本号 (MINOR)**：向下兼容的功能性新增
- **修订号 (PATCH)**：向下兼容的问题修正
